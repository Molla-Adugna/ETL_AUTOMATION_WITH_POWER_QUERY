# 📊 ETL Automation with Power Query
## 📌 Project Overview
This project demonstrates how to automate **ETL (Extract, Transform, Load)** workflows using **Microsoft Power Query**. The goal is to streamline data preparation for analytics and reporting by eliminating repetitive manual steps and ensuring clean, structured datasets ready for visualization in **Excel** or **Power BI**.

The workflow covers:

- Data extraction from raw CSV/Excel sources
- Transformation (cleaning, unpivoting, merging, deduplication)
- Loading into structured tables for analysis and dashboards

## 🎯 Key Objectives
- Automate repetitive data preparation tasks
- Ensure data integrity with consistent formatting and validation
- Enable faster reporting and visualization in Power BI
- Showcase practical ETL automation for recruiters and collaborators

## 🛠 Tools & Skills Demonstrated

| Tool/Skill        | Purpose |
|-------------------|---------|
| **Power Query**   | ETL automation (extract, transform, load) |
| **Excel**         | Data preprocessing and validation |
| **Power BI**      | Visualization and dashboarding |
| **Data Cleaning** | Remove duplicates, handle missing values |
| **Unpivoting**    | Reshape wide tables into long format |
| **Merging Queries** | Combine multiple sources into one dataset |



## 📂 Workflow Steps
**1. Extract**

- Import raw CSV/Excel files into Power Query.
- Connect multiple sources seamlessly.

**2. Transform**

- Clean column headers and remove blank rows
- Standardize date formats and numeric fields.
- Unpivot tables for normalized structure.
- Merge queries to consolidate datasets.

**3. Load**

- Output structured tables into Excel or Power BI.
- Refresh automatically with one click for updated reports.
